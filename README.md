# homebrew-tap

Homebrew tap for [ClaudeNotch](https://github.com/rawsun007/claude-notch).

## Install

```sh
brew install --cask rawsun007/tap/claudenotch
```

ClaudeNotch is ad-hoc signed (not yet notarized). On first launch, right-click ClaudeNotch in /Applications and choose Open. Then open the Setup window from the menu-bar bell to wire up the Claude Code hooks.
